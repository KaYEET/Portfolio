Over mij

Mijn naam is Kai en ik ben student. Ik ben een programmeur met een passie voor fitness, auto's en muziek. Mijn specialiteiten zijn Python en Java.

Vooraleer ik verder ging studeren aan een hoge school had ik op het middelbaar al toegepaste informatica gestudeerd. Hierbij heb ik de inleiding tot programmeren geleerd. Na deze opleiding heb ik 2 jaar toegepaste informatica gevolgd op KdG hogeschool maar ik wou me meer verdiepen op het programmeren. Hierdoor ben ik dus in Thomas More beland in het graduaat programmeren. Als je meer over mijn werkervaringen en studies te weten wil komen, kan je mijn cv bekijken of je kan een kijkje nemen op mijn GitHub profiel.

Voor geschiedenis

Ik ben opgegroeid met een computer dus ik was er ook wel al van jongs af aan mee bezig. Ik had altijd wel een interesse in hoe een applicatie of website werkte. Toen heb ik mezelf ook python aan geleerd en ik vond dit super om mee te werken en dan ook zelf programmas kon maken.
