Java Game

Wat is het?

Het is een game waarin een misdaad is gepleegd. Een moordenaar is nog vrij aan het rondlopen en je moet ondervinden wie de moord heeft gepleegd. Het spel is een text based game gebaseerd op het spel World Of Zuul.

Doel van het spel?

Je moet als speler door het spel heen reizen en verschillende locaties af gaan. Je moet te weten komen wie de moord heeft gepleegd en met welk wapen. Pas op dat je niet zelf sterft.
